# Glosario Alcohn

- "Sello": pieza de bronce grabada en CNC con el diseño del cliente.
- "Vectorizar": convertir una imagen (foto/dibujo) en un archivo vectorial apto para CNC.
- "Mockup": imagen previa de cómo quedaría el sello o el marcado.
- "Material a marcar": superficie donde el cliente quiere usar el sello (cuero, madera, alimento).
- "Seña": pago inicial para empezar a fabricar.
