# Assets y contenido faltante para elevar Alcohn

Referencia visual: DOSS usa grillas técnicas, módulos claros, paneles oscuros de producto y mucho control visual. Para Alcohn, la misma lógica funciona si reemplazamos "software" por proceso real: logo, muestra, CNC, bronce, prueba de marca y pago online.

## Tomas recomendadas

1. **Hero atelier industrial**
   - Plano horizontal, 16:9.
   - Banco de trabajo real, herramientas, bronce, cuero y madera.
   - Una persona trabajando o revisando, sin posar demasiado.
   - Luz natural/lateral, tonos negro, bronce, madera y acero.

2. **CNC / alta tecnología**
   - Video vertical y horizontal de la máquina trabajando.
   - Primer plano de viruta, herramienta, placa de bronce y movimiento.
   - Evitar planos demasiado sucios o con fondo desordenado.

3. **Antes / durante / después**
   - Logo original en pantalla.
   - Mockup digital.
   - Sello de bronce terminado.
   - Marca aplicada sobre producto final.

4. **Oficios**
   - Marroquinero marcando cuero.
   - Carpintero marcando tabla/pieza.
   - Cuchillero con hoja o funda marcada.
   - Emprendedor preparando packaging.
   - Gastronómico usando sello sobre pan, hielo o alimento si aplica.

## Nota de dirección de arte

La estética buscada no debería parecer "lujo decorativo"; tiene que sentirse como una marca premium de taller: precisión, herramientas reales, materiales nobles, grillas técnicas, mucho aire, negro/bronce/papel, y movimiento sutil que ayude a entender el proceso.

## Nueva capa visual: manual técnico / hoja de taller

La referencia de hoja impresa con anotaciones funciona muy bien para Alcohn si se usa como lenguaje de "orden de fabricación" y "manual de uso", no como decoración general. Conviene aplicarla en piezas que reduzcan dudas antes de comprar.

## Fotos prioritarias para la explicación de uso

1. **Cuero frío con remachadora**
   - Plano medio de prensa/remachadora, sello colocado y pieza de cuero.
   - Una foto del resultado en bajo relieve.

2. **Cuero caliente**
   - Sello calentándose y marca final sobre retazo.
   - Mostrar diferencia entre marca correcta y quemada.

3. **Madera caliente**
   - Sello perpendicular sobre tabla.
   - Resultado final con textura de veta visible.

4. **Packaging/cartón**
   - Prueba de baja temperatura o presión leve.
   - Marca sobre caja, etiqueta o bolsa.

5. **Alimentos**
   - Foto cuidada, limpia y específica del uso real.
   - Aclarar material, temperatura y tiempo de contacto.

6. **Kit incluido**
   - Sello, mango, adaptación para remachadora, guía de uso y packaging.
   - Esta imagen debería repetirse en producto, checkout y post-compra.
