# Checkout y pagos

## Objetivo
Permitir cobrar sin intervención de vendedor.

## Dos modalidades

### 1. Productos estándar
- Abecedarios
- Sellos de medida fija
- Accesorios
Estos pueden ir a un checkout directo.

### 2. Sellos personalizados
- El usuario paga una seña (por ejemplo $10.000).
- Se genera un pedido interno con los datos del logo.
- El resto se abona cuando el sello está listo.

## Consideraciones
- Mostrar siempre el costo de envío estimado.
- Mostrar tiempos de entrega.
- Confirmar por mail/whatsapp el pedido.
