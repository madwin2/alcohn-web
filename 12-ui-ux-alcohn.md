# UI / UX Alcohn

## Estilo visual
- Industrial minimal / high-tech.
- Paleta: grises, negro, blanco + acento flúo (amarillo/verde).
- Tipografía sans técnica.
- Espacios amplios, estética de taller profesional.

## Componentes que se repiten
- Hero con CTA doble.
- Cards de materiales.
- Timeline de pasos.
- Testimonios.

## Accesibilidad
- Botones claros.
- Texto explicativo en cotizador.
- Versión mobile prioritaria.
