# Productos

## Objetivo
Que el usuario entienda que puede pedir un sello según el material o el uso. Esto ordena la cabeza del cliente y facilita campañas.

## Categorías
1. Sellos para cuero
2. Sellos para madera
3. Sellos para alimentos
4. Abecedarios de bronce
5. Accesorios (mangos, soportes, prensas recomendadas)

## Estructura de card de producto
- Foto del sello real
- "Para material: cuero/madera/alimentos"
- Medidas más usadas
- Precio "desde" (ej: "desde $44.000")
- Botón: "Quiero este sello con mi logo" → `/cotizar?tipo=cuero` (params)

## Notas
- La página de producto NO tiene que pedir el logo ahí mismo.
- Solo dispara el flujo de cotización.
