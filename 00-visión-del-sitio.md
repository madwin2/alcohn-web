# Visión del sitio: Alcohn

## Objetivo principal
Crear un sitio web que funcione como un vendedor online de los sellos de bronce de Alcohn. Debe informar, generar confianza y permitir iniciar/completar una compra sin depender de un vendedor humano.

## Problema actual
- Muchas consultas entran por WhatsApp/Instagram.
- Los vendedores pierden tiempo explicando lo mismo.
- Hay clientes que no terminan comprando porque no saben el proceso.

## Solución
- Web con estructura clara por tipo de uso.
- Página de proceso para explicar "cómo funciona".
- Calculadora + subida de logo para empezar el pedido.
- Checkout para pagar seña o productos estándar.
- Integración de WhatsApp solo para casos especiales.

## Resultados buscados
1. Bajar el tiempo de los vendedores.
2. Aumentar la conversión de visitas en leads/compras.
3. Posicionar a Alcohn como marca líder, tecnológica y profesional.
